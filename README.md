### About

I build intelligent, scalable platforms at the intersection of **AI**, **data engineering**, and **modern full-stack development** — designing multi-agent systems, RAG pipelines, and production-grade LLM applications, backed by robust data infrastructure and clean, performant UIs.


### What I'm Working On

- Architecting **agentic AI systems** with LangGraph, LangChain, RAG pipelines, and custom tool integrations
- Building large-scale ingestion and analytics pipelines on **Databricks** — Delta Lake, PySpark, Unity Catalog, Databricks SQL
- Shipping full-stack applications with **React, Next.js, TypeScript, and Python** wired to AI-backed APIs
- Deploying secure, cloud-native services on **Azure** — App Services, Blob Storage, CosmosDB
- Contributing to **open source** across the LLM and developer tooling ecosystem
